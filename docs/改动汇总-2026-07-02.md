# PanGrab Pro 改动与新增汇总（截至 2026-07-02）

> 本文档汇总本轮全部改动，供备份查阅。仓库 `HveinBlg/pangrab-pro`，主分支 `main`，最新提交 `5f137de`。
> 统计：27 个文件变更，约 +2321 / -386 行。

---

## 一、按功能模块汇总

### 1. UI 美化
- **自动同步开关**：原生复选框 → 滑动开关（Toggle）；文字左、开关右，修复被 `.modal-card label` 样式覆盖的错位
- **会员徽章**：绿色药丸 → 金色渐变 + 皇冠高级质感，右对齐；邮箱/有效期左列
- **收藏卡片布局**：标题 → 链接(+提取码同行) → (封面图在左 + 简介在右)；标题/简介精简（标题≤40字2行，简介≤80字3行）
- **封面统一**：固定长方形 160×100（定高 + 居中裁切），各处比例一致

### 2. 国际化 i18n（简 / 繁 / 英，自动切换）
- 新增 `_locales/{en,zh_CN,zh_TW}/messages.json`（各 222 键）
- 新增 `src/i18n.js` 运行时（HTML `data-i18n` 自动填充 + JS `t()` 取词）
- `manifest.json` 加 `default_locale`，名称/描述用 `__MSG__`
- 覆盖 popup、options、pro.js 账号面板、background 通知、manifest
- 购买页 `buy.html` 用内置 JS 字典按浏览器语言（zh/tw/en）切换

### 3. 支付与收款
- 购买页按**时区 + 语言**判断地区：大陆→支付宝，台湾/香港/海外→国际支付(Lemon Squeezy)
- 购买按钮文案按地区自适应
- `.env.example` 补全 Lemon Squeezy 配置逐步指引

### 4. 管理后台重做（server/src/admin.html）
- 界面美化：渐变顶栏、彩色统计卡、标签页（用户 / 兑换码）
- **新增用户列表**：显示每个账户的邮箱、会员状态、到期、**地区(带国旗) + IP**、最近登录、登录次数、注册时间；支持搜索/筛选
- 后端配套：新增 `/api/admin/users`；登录/注册记录 IP + 地区（`geoip-lite` 离线反查）；`trust proxy`；users 表加字段迁移

### 5. 磁力链接（magnet）功能 ★本轮重点
- **识别**：`magnet:?xt=urn:btih:...`，名称取自 `dn`
- **去重合并**：按 btih 哈希（同资源不同 tracker/名称算一条）
- **会员专属**：非会员检测到但不能收藏，提示开通
- **按番号自动分组**：从文件名(dn)解析番号（归一化为 `字母-数字`，如 HMN-208），同番号的多个磁链自动归入同一分类"文件夹"
- **封面 + 简介**：通用 og 元信息 + 页面最大图兜底；磁链简介优先显示"番号：xxx"；网盘只取封面、不显示简介；裂图优雅隐藏
- **失效检测**：磁链跳过（非 HTTP 无法检测）

### 6. 独立「番号库」页面（src/folders/）
- 番号做成**文件夹网格**（封面 + 番号 + 数量），点进去看该番号下所有磁链
- 详情为自适应多列网格；支持搜索番号、返回；数据变化自动刷新
- 番号自动分类**不再挤在**普通分类下拉/筛选里（避免番号过多刷屏），统一在此页面查看

### 7. 文档
- 更新 `README.md`（完整功能）、修正 `DEPLOY.md` 克隆地址
- 新增 `docs/开发复盘-2026-07.md`、`docs/服务器迁移指南.md`、本汇总

## 四、部署提醒
- **扩展**：`git pull` 后在 `chrome://extensions` 重新加载扩展（多数改动只在扩展端）
- **后端**（仅当动了 server/）：`git pull` → `npm install`（新增 geoip-lite）→ `pm2 restart pgpro`
- **旧数据**：番号分组/封面/简介为新增字段，仅对更新后**新收藏**的生效；旧磁链需重新收藏或手动归类
- 详见 `docs/服务器迁移指南.md`

_对应 `main@5f137de`。_
