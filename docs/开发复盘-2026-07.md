## 二、i18n 国际化方案（重点）

### 目标
界面按浏览器语言**自动切换**：简体中文 / 繁体中文 / English。不是"要么全中文要么全英文"，而是各看各的语言。

### 架构
- **扩展部分**用 Chrome 官方机制：`_locales/{en,zh_CN,zh_TW}/messages.json` + `chrome.i18n`
  - `manifest.json` 设 `default_locale: "en"`，name/description/title 用 `__MSG_key__`
  - 新增 `src/i18n.js` 运行时：
    - HTML 元素加 `data-i18n` / `data-i18n-ph` / `data-i18n-title` / `data-i18n-label` / `data-i18n-html` 属性，页面加载自动填充
    - `<html data-i18n-doctitle>` 设置文档标题
    - JS 里用 `t("key", sub1, sub2)` 取词（等价 `chrome.i18n.getMessage`，占位符用 `$1/$2`）
  - Service Worker（background.js）直接用 `chrome.i18n.getMessage`（无法用 i18n.js）
- **购买页 buy.html** 是服务端页面，**不能用 chrome.i18n**，改用内置 JS 字典（`DICT{zh,tw,en}` + `L(key)`），按 `navigator.language` 选择

### 覆盖范围
popup、options（收藏管理页）、pro.js（账号/云同步面板）、background 通知、manifest、buy.html 全部文案。

### 有意保留中文的两处（非遗漏）
1. **网盘品牌名**（百度网盘、夸克…）：品牌专名，且已写入用户存量数据，翻译会破坏数据
2. **失效检测关键词** `DEAD_KEYWORDS` 正则：用于匹配中文网盘"已失效"页面内容，必须是中文才命中
3. 另：内部分类哨兵值 `"未分类"` 作为跨语言数据标识保留，显示时用 `t("cat_uncategorized")` 本地化

### 语言匹配规则
- 简体 `zh-CN` → zh_CN；繁体 `zh-TW/zh-HK/zh-Hant` → zh_TW；其它 → en（默认）
- 校验：三个语言包各 **213 键完全一致**，代码引用的键全部存在

### 如何测试
Chrome → 设置 → 语言 → 把目标语言拖到最上并"以此语言显示" → 重启浏览器 → 扩展随之切换。

## 三、支付与收款（现状与决策）

### 双渠道设计
| 渠道 | 面向 | 资质 | 收款 | 代码状态 |
|------|------|------|------|---------|
| 支付宝当面付 | 国内用户 | 国内营业执照（个体工商户即可）| 人民币 | 已集成，已配置可用 |
| Lemon Squeezy | 海外用户 | 新加坡 OCBC 账户 | SGD（银行转账）| 已集成，**待后端配置** |

购买页地区判断会自动分流；两个都配则通吃国内外。

### 关键认知
- **国内支付宝收不了普通海外用户的钱**（对方需有支付宝账号），海外必须靠 Lemon Squeezy（信用卡/PayPal/本地货币）
- Lemon Squeezy 已被 Stripe 收购，开店走 Stripe 实名；用**中国护照**可完成身份验证
- 已确认：用**新加坡本地 OCBC 账户**收款（分行 PEOPLE'S PARK [624]，360 SGD 账户）→ 可走银行提现，优于 PayPal
- 买家来自哪个国家不受收款账户限制——LS 负责换汇+代缴税，汇总打 SGD 到 OCBC

### Lemon Squeezy 待办（审核通过后）
后端 `server/.env` 需填 6 项（详见 `server/.env.example`）：
```
PUBLIC_BASE_URL、LEMONSQUEEZY_API_KEY、LEMONSQUEEZY_STORE_ID、
LEMONSQUEEZY_WEBHOOK_SECRET、LEMONSQUEEZY_VARIANT_MONTH/_QUARTER/_YEAR
```
- 商品务必建为 **One-time 单次付款**（代码只监听 `order_created`，订阅事件不会开通会员）
- Webhook URL：`<PUBLIC_BASE_URL>/api/lemonsqueezy/webhook`，勾选 `order_created`
- API Key 分 Test/Live，勿混用

### 当前线上状态
- 支付宝已配好：国内用户可扫码付款、自动开通
- Lemon Squeezy 未配：购买页只显示支付宝（符合预期）；海外用户暂用**兑换码**顶替
